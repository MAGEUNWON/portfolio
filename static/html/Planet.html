  • 랜덤 영화 추천 사이트

    볼게 너무 많은 요즘!
    뭘 볼지 고민하기도
    귀찮은 사람들을 위해
    장르별로 영화를
    랜덤 추천해주는 서비스!

    개발 기간
    22.09.06~22.10.27

    개발 인원: 3명

    역할(팀원)
    -랜덤 페이지(기여도 90%)
    -회원가입 페이지(기여도 80%)
    -마이페이지-내 정보 수정(기여도 100%)
    -게시판 글 작성 화면(기여도 90%)

    언어: HTML, SCSS/CSS, JavaScript
    서버: Node.js
    DB: MySQL
    IDE: Visual Studio Code, MySQL WorkBench
    Framework: Express.js, Figma
    API : KMDB, 영화진흥위원회
    협업 : git, slack

  • 1. 장르 버튼 누르면 영화를 장르별로 랜덤 추천해서 보여 줌
    2. 영화 포스터를 클릭하면 카드가 뒤집히면서 영화의 간략한 설명이 나옴
    3. 다른 영화 추천 받기를 누르면 다시 랜덤으로 영화가 나옴

  • 1. 메인 -> 로그인 -> 회원가입을 누르면 가입 작성 화면이 나옴
    2. 회원가입란 작성 후 가입하기 버튼을 클릭하면 가입 완료
    3. 회원가입시 적은 정보로 로그인 하면 로그인 완료

  • 1. 메인 -> 마이페이지 -> 내 정보 수정에서 프로필 이미지 등록 가능
    2. 유효성검사로 올바른 비밀번호를 입력할 수 있도록 경고창이 뜸
    3. 탈퇴하기 버튼 누르면 모달창으로 확인 여부가 나옴

  • 1. 메인 -> 게시판 -> 글쓰기 누르면 글 작성 할 수 있는 화면이 나옴
    2. 등록하기 버튼을 누르면 글이 등록되고 목록으로 버튼을 누르면 다시 게시판 화면으로 돌아감
